# Tire Shop Management System

Full-stack inventory management system for tire shops.

## Tech Stack
- Frontend: React + Vite
- Backend: FastAPI + Python
- Database: PostgreSQL

## Local Development

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Login: admin / admin
